Remote Calibrator

Calibrate screen size and track head and gaze for online experiments.

Programmers: Remote Calibrator package can build the step-by-step calibration panel at the start of the experiment with a few lines of code. This page runs in demo mode. In a real experiment, participants would only be allowed access to one step at a time. All the text, including the headline, the description, and the next-step button, can be customized.
