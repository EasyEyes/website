The open-source EasyEyes web app is meant to help anyone testing vision online. EasyEyes measures screen size once and uses the webcam to track head and gaze continuously. It also allows use of a smartphone to emulate a wireless keypad. Currently, we have a working prototype with limited documentation. We are using it with PsychoPy. You are welcome to use it with any online testing software.
